# Launch Checklist

Single unified checklist for getting VidyaRas dashboard ready for public app launch.

**File:** `launch_checklist.md`

**Focus:**
- API layer for both dashboard and app
- Dashboard features matching current app capabilities
- No analytics or advanced features (those come post-launch)

**Current App Features Supported:**
- Courses with lectures (organized by sections)
- Live classes
- Course materials
- Quizzes (text/LaTeX/image content support)
- Student enrollments
- Progress tracking

**Start Here:**
1. Phase 1: Build API layer
2. Phase 2: Build dashboard frontend
3. Phase 3: Test everything
4. Phase 4: Deploy

Check off tasks as you complete them.
